Interactive HTML maps showing **storm surge + estimated wind setup** inundation across all 14 parishes of Jamaica for a modelled Hurricane Melissa event.

## Scenario Description

- **With estimated setup** — includes the base storm surge plus an estimated wind setup contribution, giving a higher inundation estimate than surge alone.
- Compare with [`../Hurrican_Melissa_surge_only/`](../Hurrican_Melissa_surge_only/) to see the added contribution of wind setup, and with [`../Hurrican_Melissa_with_rain/`](../Hurrican_Melissa_with_rain/) to see the rainfall scenario.
