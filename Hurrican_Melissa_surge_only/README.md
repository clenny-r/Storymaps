# Hurricane Melissa — Storm Surge Only

Interactive HTML maps showing **storm surge inundation** across all 14 parishes of Jamaica for a modelled Hurricane Melissa event, considering **surge only** (no rainfall or additional setup contributions).

## Scenario Description

- **Storm surge only** — shows coastal inundation from the storm surge driven by Hurricane Melissa's winds and pressure, without accounting for rainfall runoff or wind setup contributions.
- Use alongside [`../Hurrican_Melissa_with_est_setup/`](../Hurrican_Melissa_with_est_setup/) and [`../Hurrican_Melissa_with_rain/`](../Hurrican_Melissa_with_rain/) for a full picture of combined hazard scenarios.
