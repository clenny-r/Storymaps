## Scenario Description

- **With rain** — the most comprehensive scenario, combining storm surge inundation with the additional flooding contribution from Hurricane Melissa's rainfall. This typically represents the worst-case inundation extent.
- Compare with [`../Hurrican_Melissa_surge_only/`](../Hurrican_Melissa_surge_only/) and [`../Hurrican_Melissa_with_est_setup/`](../Hurrican_Melissa_with_est_setup/) to isolate the contribution of each component.
