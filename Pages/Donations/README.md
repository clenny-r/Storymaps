# Donations

This folder contains the giving pages for TWC Ministries, with separate pages for local (Jamaica) and foreign (international) donors.

**Live URL base:** `twcministries.net/Pages/Donations/`

## Files

| File | Description |
|---|---|
| `twc-ministries-local-account.html` | Giving instructions for local (Jamaican) donors |
| `twc-ministries-foreign-account.html` | Giving instructions for international donors |

## Notes

- Each page displays the relevant bank account details and payment options for its audience.
- These pages do not process payments directly — they provide account information for manual transfers.
- **Do not commit sensitive account credentials directly into these files.** If account numbers are displayed, ensure the repo visibility matches your privacy needs.
