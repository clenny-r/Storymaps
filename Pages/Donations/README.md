# Donations

Giving pages for TWC Ministries, split by donor location.

## Subfolders

| Folder | Live URL | Description |
|---|---|---|
| `local/` | [/Pages/Donations/local/](https://www.twcministries.net/Pages/Donations/local/) | Local (Jamaican / JMD) giving instructions |
| `foreign/` | [/Pages/Donations/foreign/](https://www.twcministries.net/Pages/Donations/foreign/) | International (USD) giving instructions |

Each folder contains an `index.html` with the relevant bank account details and payment options. These pages display account information for manual transfers and do not process payments directly.
