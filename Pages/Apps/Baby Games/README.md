# Baby Games

Bible-themed interactive games designed for young children at TWC Ministries.

**URL base:** `twcministries.net/Pages/Apps/Baby Games/`

## Games

| File | Description |
|---|---|
| `baby-squish-game.html` | A simple squish/tap game for toddlers |
| `baby-bible-squish.html` | Bible-themed variant of the squish game |

## Notes

- Both games are single-file HTML pages — no dependencies or build step needed.
- Designed for touchscreens (tablets/phones) so children can play independently during services.
- The folder name contains a space (`Baby Games`). In URLs this becomes `Baby%20Games`. Consider renaming to `Baby_Games` to avoid encoding issues.
